# Vest.IA 👕🤖

## 🎯 Descrição do Problema
Muitas pessoas têm dificuldade em organizar seus guarda-roupas, esquecem as peças que possuem ou perdem muito tempo decidindo o que vestir. O gerenciamento manual de roupas é trabalhoso e pouco eficiente.

## 🚀 Objetivo
Criar um sistema inteligente (Vest.IA) que atue como um guarda-roupa virtual. O aplicativo permite catalogar roupas de forma automatizada utilizando Inteligência Artificial (Google Gemini) para identificar características das peças a partir de fotos, facilitando a organização e o uso diário.

## ✨ Principais Funcionalidades (Casos de Uso)
1. **Cadastro Inteligente de Roupas:** O usuário faz o upload de uma foto da peça e a IA extrai automaticamente os dados (tipo, cor, ocasião, clima ideal).
2. **Listagem do Guarda-Roupa:** Visualização de todas as roupas cadastradas no banco de dados.
3. **Registro de Uso:** Sistema que contabiliza quantas vezes uma roupa foi usada e guarda o histórico de datas.
